# Installation

## Homebrew (macOS)

```bash
brew tap withakay/spool
brew install spool
spool --version
```

## Prebuilt Binary (macOS/Linux)

```bash
curl -fsSL https://raw.githubusercontent.com/withakay/spool/main/scripts/install.sh | sh
spool --version
```

## Build From Source

```bash
make rust-install
spool --version
```

## npm (optional)

```bash
npm install -g @withakay/spool
spool --version
```
