# Quick Start Prompts

Copy and paste these prompts to your AI assistant to get started with Spool.

## Set up your project

Read .spool/project.md and help me fill it out. Detect the tech stack, purpose, and architecture from the codebase.

## Define coding standards

Read .spool/user-guidance.md and help me document project-specific coding standards. Look at existing code patterns and suggest conventions.

## Create a new feature

I want to add a new feature. Help me create a spool change proposal.

## Fix a bug

I found a bug: [describe the bug]. Help me create a spool change to fix it.

## Refactor code

I want to refactor [describe what]. Help me create a spool change proposal with a clear scope.

## Review changes

Run `spool list` and help me review the status of active changes.

## Continue work on a change

Run `spool agent instruction apply --change [change-id]` and help me continue implementing it.
