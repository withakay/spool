# Spool Workflow Instructions

When working on a Spool change, get the canonical workflow instructions from the CLI:

- **Proposal/specs/design/tasks**: `spool agent instruction proposal|specs|design|tasks --change "<id>"`
- **Implementation**: `spool agent instruction apply --change "<id>"`
- **Review**: `spool agent instruction review --change "<id>"`
- **Archive**: `spool agent instruction archive --change "<id>"`

Do not embed long, duplicated workflows here.
