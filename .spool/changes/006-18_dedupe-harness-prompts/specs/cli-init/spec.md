## ADDED Requirements

### Requirement: Harness instruction wrappers delegate to `spool agent instruction`
`spool init` SHALL install harness-specific prompt/command wrappers whose managed body delegates to `spool agent instruction <artifact>` so the canonical instruction content is generated by the CLI.

#### Scenario: OpenCode command wrapper uses instruction artifacts
- **WHEN** `spool init` configures OpenCode commands
- **THEN** each generated `.opencode/commands/spool-*.md` managed block SHALL instruct the agent to obtain the full workflow instructions by running `spool agent instruction <artifact> --change <change-id>`

#### Scenario: GitHub Copilot prompt file wrapper uses instruction artifacts
- **WHEN** `spool init` configures GitHub Copilot prompt files
- **THEN** each generated `.github/prompts/spool-*.prompt.md` managed block SHALL instruct the agent to obtain the full workflow instructions by running `spool agent instruction <artifact> --change <change-id>`
